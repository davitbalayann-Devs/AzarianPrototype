The About page Hero (`Azarian About.dc.html`, `.agn-about-hero`) is a full-bleed dark-purple section built from Figma node 22739-2677 ("Grid squares"). A JS-generated CSS grid of 144px transparent squares (`setupHero()` in the component script) fills the viewport width; team photos (`assets/team/hero/hero-01..08.png`, downscaled to 320px) are scattered on non-adjacent cells, kept out of a center "safe zone" so they never sit behind the heading/subtext.

Key behaviors: hero is full-screen (`min-height:100vh/100dvh`). Square cells RESIZE with the screen — cell = min(gridWidth/baseCols, room-above-text, room-below-text/2.1), then cols retiled to fill width exactly; the column count differs per screen but the composition is constant. This vertical capping guarantees a row fits above the text and two below it so ALL 8 photos stay fully visible at any size. The grid is offset by `gTop = (navClear % cell) - cell` so it bleeds a little past the top edge (fading in behind the nav) while a row boundary still lands exactly at `navClear` (≈112px) for the top band, which therefore never tucks under the overlaid nav. Photos are placed DETERMINISTICALLY (never random positions): three column-fraction bands — top `[.14,.52,.86]`, lower `[.14,.52,.86]`, bottom pair `[.34,.66]` — at rows computed as firstAbove/firstBelow the measured text rect, flanking the vertically-centred text with no overlap. Text widths match Figma: heading `max-width:1296px`, subtext `max-width:932px`. Grid is full-bleed (fills the whole viewport width, no 1920 cap) and fades on ALL FOUR edges via a CSS `mask-image` — horizontal `7%/93%`, vertical `108px` (top, so the top band stays crisp) `/ 93%` (bottom) — combined with `mask-composite:intersect`. Photos rest at `opacity:.2`; hover → opaque + radial-gradient backing + accent border (Figma hover node 22739-2720). Intro: squares sweep in (left half bottom→up, right half +20ms top→down) → heading char-by-char SplitText (delay 50ms, dur 1.25s, power3.out) → photos blur-in in random ORDER (positions still fixed). Honors `prefers-reduced-motion`; rebuilds on resize without replaying the intro. See [[azarian-site-architecture]].

The hero uses the same parallax cover-scroll as the Home page: it's wrapped in `.agn-hero-scroll` (`z-index:0`, 100dvh) with a 100dvh `.agn-hero-spacer`; the hero header is `position:fixed` (`data-hero-pin`), and `.agn-hero-scroll ~ *{z-index:2}` makes the following sections rise over the pinned hero on the first scroll. `setupHeroCover()` (mirrors Home's method) drives it on scroll/resize via rAF: progress `raw = -scroll.top / vh`, easeOutCubic → hero gets `scale 1→1.22`, `translateY→28px`, `--hero-veil 0→.55` (dark `::after` overlay), `blur 0→16px`, `opacity 1→.65`, `visibility:hidden` at end; the content layer (`data-hero-layer`) fades faster. Disabled under `prefers-reduced-motion` (hero reverts to `position:relative`). The page root div's background is the hero gradient with `background-attachment:fixed` (not white) so the pinned hero, as it fades to partial opacity during the cover-scroll, blends into the same dark gradient instead of washing out to the white page background.

## 02 About Hamlet section
Directly below the hero (`section#story.agn-hamlet`, replaces the old "Our story" section, Figma node 22447-8021). Dark `#021526` bg with a blurred purple radial glow; left = Hamlet's transparent cut-out photo (`assets/team/hamlet-round.png`) on a `#564bcc` circle with a glass rim; right = text column (two white body paragraphs, a teal `#1bfed1` accent line, a bold white lead). `setupHamlet()` in the component script implements the BlurText reveal in vanilla WAAPI: each `[data-blur]` paragraph is split into `.agn-word` spans that animate blur(10px)→blur(5px)→blur(0) / opacity 0→.5→1 / translateY(-50px)→5→0 (dur 700ms, linear), cascading top→bottom across all paragraphs in turn (≈14ms per word + 160ms between blocks); the photo blurs in first. Triggered once via IntersectionObserver (threshold .1). Hero team photos live in `assets/team/hero/` named by person (Amile, Antika, Arevik, Davit, Hamlet, Mari, Naneh, Sara).
